System - Privacy Policy<

Privacy Policy:

1. -System does not store any user data.

2. -System does not store any message content.

3. -The System Staff is able to see the names and id's of the servers that use System.

4. -The staff is not able to read the content of the direct messages sent to System.

5. -No messages will be redirected to any 3rd party members.

6. -When using commands like ip_info, no ip's / urls / etc. will be stored.
